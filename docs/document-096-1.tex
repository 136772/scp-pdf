\bb{实验096-1的文件记录\#096-1}

\ii{实验096-1由Dan博士主持，目的是在获取完整的\hyperref[chap:SCP-096]{SCP-096}身体特征描述的同时测试SCP-096的能力。}

\ii{D-9031是一名32岁的已定罪重刑犯，也是一位前纹身艺术家。D-9031被安置在球形潜水舱303-A中，之后潜水舱沉入新西兰海岸附近的汤加海沟（Tonga Trench）。该位置距离██号站点中的SCP-096临时收容隔间约███公里。球形潜水舱303-A与Dan博士位于新西兰本岛的控制站之间的以下交流通过潜水舱中的视频监控记录。}

\ii{球形潜水舱303-A到达最终深度10,800米处。}

\bb{D-9031：}停下来了。现在干嘛？

\bb{Dan博士：}你感觉还好吗？没觉得恶心什么的吗？

\bb{D-9031：}我耳朵疼。

\bb{Dan博士：}那是正常现象。现在，你左边应该有一个钢制容器。打开它，里面有一个装着几张照片的马尼拉纸文件夹。请打开文件夹，描述第一张照片。

\ii{D-9031照做了。摄像机是固定着的，因此看不见照片。}

\bb{D-9031：}什么都没有。一个空房间。

\bb{Dan博士：}谢谢你。请把这张照片正面朝下放到你右边的容器里，然后看下一张。

\bb{D-9031：}同一个房间，但是……有只脚在里边，我觉得。

\bb{Dan博士：}请描述它。

\bb{D-9031：}呃……苍白，皮包骨头。说实话有点吓人。

\bb{Dan博士：}把照片放进容器，正面朝下，再看下一张。

\bb{D-9031：}Ok……{[}停顿]我操！

\bb{Dan博士：}描述照片。

\bb{D-9031：}这是个……我说不准，一个怪可怕的人。

\bb{Dan博士：}请描述照片。

\bb{D-9031：}见鬼了。他惨白惨白的，白眼睛，他妈嘴巴也不对劲。这鬼东西到底是个啥？

\ii{此刻，标准时间大约13:32，Dan博士和实验控制处接到了SCP-096突破收容的通报。平民和图像捕捉设备已被从到SCP-096-1的最快路径上移除。SCP-096现正通过发信项圈由卫星追踪。}

\bb{Dan博士：}你的右边应该有另一个钢制容器。打开它。

\bb{SCP-096-1：}是一个便笺簿和一支铅笔。

\bb{Dan博士：}是的。请画出你看到的照片的速写。

\ii{SCP-096-1小声骂了一句，用接下来的二十分钟画出了照片的速写。速写完成时，确认SCP-096距离SCP-096-1███公里。}

\bb{SCP-096-1：}画完了。

\bb{Dan博士：}很好。把画放到你左边的容器里，关上门。

\ii{SCP-096-1照做了，速写装在一个防水可漂浮容器中离开了球形潜水舱303-A。其它照片随后在舱内的焚化炉内被焚毁。}

\bb{SCP-096-1：}现在呢？

\bb{Dan博士：}请待命。

\ii{四十分钟过去了。现确认SCP-096位于SCP-096-1的所在地，正在下潜。应答器信号在9,339米深处由于水压超过了设备的极限而中断。摄像机显示潜水舱轻微晃动。从SCP-096-1的反应来看，可假设SCP-096在舱体上并能从观察孔看到。}

\bb{SCP-096-1：}我操！操操操！见鬼那是啥？！

\ii{视频和音频传输因为球形潜水舱303-A的舱体被打破而中断。SCP-096由水面回收组Foxtrot-303-A平安回收。SCP-096的速写也被回收了，快速测试确认SCP-096未被激发敌意反应。}

\ii{SCP-096被移至永久收容处，与此同时速写被送往新西兰的实验控制处。}
