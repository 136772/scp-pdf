## LaTeX

## 字体

中文使用 Noto Sans CJK SC

英文使用 Noto Sans

中文等宽使用 Noto Sans Mono CJK SC

英文等宽使用 Fira Code

Emoji 使用 Noto Emoji

## 帮忙

按照相同的格式排版一篇试试看……我弄了一些 macro 之类的来辅助排版，常用的格式应该都没问题了。

如果是第一次 PR 需要看一眼 editors.tex 和 version.tex，加上自己。

然后 PR 到 dev 分支，应该就行了。
